Technical Portfolio

Client Server Connection

In my Introduction to Computer Systems Class, I created a connection between Client and Server using multiple different connections like a FIFO channel and a TCP channel. The client could request data points from the server as well as perform file transfers from the server. Throughout the class, I continually worked on this client server project. With the introduction of threads, the program had a speedup time by using threading. At the end of the class, I created a TCP connection that allowed the client and server to be hosted on two different PC's. The TCP connection was created using sockets to connect to the server. For my project, I ended up hosting the client and server on virtual machines hosted on Google's cloud to connect them between one another.

Linux Shell

For this project, I recreated the Linux shell using exec functions. Commands like grep, ls, and echo could be used in this shell. The program used a lot of string manipulation to extract commands from user input. The shell also allowed for input/output redirection so the program could place the output of a command into a text file. For this project, I used pipelining to allow the output of a command to be used as the input of the next command. For the pipelining to work, I used a fork to create child processes that executed commands. At the end of the program, I used waitpid to kill zombie processes.

String Calculator

In this project, I created a calculator that could perform addition, subtraction, multiplication, and division. The calculator took in user input as strings and performed the calculations by going through the string one character at a time until the entire calculation was finished. This method allows for the calculator to perform calculations that would not typically be able to be performed due to integer overflow in C++.
